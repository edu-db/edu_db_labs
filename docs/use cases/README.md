1. ID | User_Registration
------- | :------------
НАЗВА | Створення облікового запису користувача у системі
УЧАСНИКИ | Користувач, Система 
ПЕРЕДУМОВИ | Обліковий запис користувача попередньо не був створений у системі 
РЕЗУЛЬТАТ | Створення облікового запису користувача в системі
ВИКЛЮЧНІ СИТУАЦІЇ | UserReg_EX_AccAlreadyExists — Обліковий запис користувача вже створений у системі<br> UserReg_EX_NoInput — Користувач не ввів свої реєстраційні дані
ОСНОВНИЙ СЦЕНАРІЙ | :point_down:

@startuml
    
    |Користувач|
        start
        : Вводить реєстраційні дані;

        : Натискає кнопку 
        "Зареєструватися";

    |Система|
        : Система перевіряє наявність 
        облікового запису користувача;

        обліковий запис;

    |Користувач|
